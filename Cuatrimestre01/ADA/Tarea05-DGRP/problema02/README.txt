Para el quicksort             |     Para la particion hoare
                              |
                              
En linux o macOS                    
g++ quicksort -o prog         |     g++ hoare-quicksort -o prog
./prog < prueba[i]            |     ./prog < prueba[i]      #Donde i, puede tomar valores de 1 a 10, o la O, cuando sea el peor caso

En Windows
g++ quicksort -o prog         |     g++ hoare-quicksort -o prog
prog.exe < prueba[i]          |     prog.exe < prueba[i]      #Donde i, puede tomar valores de 1 a 10, o la O, cuando sea el peor caso
